SZTAB

Raport z Pamięci. Music first, now expanding into image.

SZTAB began as a historical memory music album: SZTAB — Raport z Pamięci. It is now expanding into cinematic album videos and a companion animated series based on selected songs, figures and historical memory themes.

Orientation

SZTAB is not only an animated series.

It is a wider historical-memory project by Vibrosław: an album foundation, cinematic visual extensions and animated strands shaped around biographies, battles and forgotten stories.

Album Foundation

SZTAB — Raport z Pamięci

The first historical memory music album by Vibrosław and the source layer from which later visual formats can grow.
Music album Active

First historical memory music album

SZTAB — Raport z Pamięci established the project as historical memory music: songs built around Polish history, resistance, conscience and remembrance. The album remains the source context for several later audiovisual directions.

Album origin Later version

Kurier Prawdy

“Kurier Prawdy” comes from the first SZTAB album and was later used in a newer version. This distinction matters: the album origin remains part of the SZTAB foundation, while later versions can open separate visual or performance contexts.
